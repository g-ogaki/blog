# Visual Language

## Overall Style

* Modern
* Refined
* Human
* Calm
* Technically sophisticated

---

# Layout

Prefer:

* generous whitespace
* strong alignment
* consistent spacing
* restrained visual density

The homepage may use larger visual spacing than blog pages.

---

# Typography

Typography is the primary visual element.

General direction:

* Medium-sized body text
* Strong hierarchy
* Comfortable reading width
* Excellent code and mathematical rendering

---

# Corners

Use slightly rounded corners consistently throughout the site.

Avoid sharp or excessively rounded components.

---

# Color

Use one signature accent color consistently throughout the site.

The majority of the interface should remain neutral.

The accent color should draw attention without dominating the page.

---

# Imagery

Images should support the writing rather than decorate it.

Homepage:

* Portrait photography
* Carefully selected supporting imagery

Articles:

* Diagrams
* Screenshots
* Technical illustrations

---

# Motion

Motion should:

* communicate quality
* improve perceived responsiveness
* reinforce hierarchy

Avoid:

* decorative animations
* continuous motion
* distracting transitions

---

# Components

## Homepage

* Hero
* About section
* Latest posts
* Footer

Homepage components may contain richer visual treatment.

---

## Blog

* Search
* Category filter
* Tag filter
* Article cards

Emphasize usability over visual expression.

---

## Articles

* Comfortable typography
* Mathematics
* Syntax-highlighted code
* Link cards
* Comments

Reading experience takes priority over decoration.

---

# Dark Mode

Dark mode should preserve the same visual language as light mode.

It is an alternative appearance rather than a different design.

---

# Accessibility

Visual refinement must never reduce usability.

Prioritize:

* readable typography
* sufficient contrast
* visible focus states
* keyboard accessibility
